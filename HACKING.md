# Building and changing the book

This documentation is written with
[gitbook](https://github.com/GitbookIO/gitbook). After
[installing](https://github.com/GitbookIO/gitbook/blob/master/docs/setup.md)
it, you can rebuild hte book with `gitbook build`. To edit it, run `gitbook
serve` to view your changes in realtime.
